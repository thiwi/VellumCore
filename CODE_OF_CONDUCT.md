# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free
experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss technical disagreements with evidence.
- Focus review feedback on code and behavior, not people.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Personal attacks, trolling, or inflammatory comments.
- Publishing private information without consent.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, or pull
requests that violate this Code of Conduct.

## Reporting

Report incidents through the security channel defined in `SECURITY.md`.
