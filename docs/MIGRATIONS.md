# Migration Guides

This index tracks major-version migration guides for Vellum Core HTTP surfaces and SDK-facing usage patterns.

## Available Guides

- [Migration v5 to v6](MIGRATION_V5_TO_V6.md)
- [Migration v4 to v5](MIGRATION_V4_TO_V5.md)

## Policy

- Every breaking release must add or update a migration guide.
- Migration guides must include endpoint mappings, payload mappings, and error-code behavior changes.
- `CHANGELOG.md` and `docs/releases/` entries must link to relevant migration guides.
