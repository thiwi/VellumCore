# Vellum Core v6.0.0

Release date: 2026-03-30

## Summary

V6 introduces a breaking, resource-oriented HTTP API for run lifecycle tracking and attestation export.

## Breaking changes

- Removed all `/v1/*` and `/v5/*` reference-service routes.
- Introduced primary v6 run resources:
  - `POST /v6/runs`
  - `GET /v6/runs/{run_id}`
  - `GET /v6/runs/{run_id}/attestation`
- Replaced v5 evidence fields with discriminated `evidence` union in run-create requests.
- Replaced flat attestation policy fields with `policy { id, version }`.

## Upgrade checklist

- Follow [Migration v5 to v6](../MIGRATION_V5_TO_V6.md).
- Run contract snapshots (`pytest -m contract`).
- Run integration tests (`pytest -m integration`).
- Verify dashboard/API clients now call `/api/v6/*` and `/v6/*` routes.
